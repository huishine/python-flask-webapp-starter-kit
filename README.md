# Python Flask Starter Kit

## Usage
1. $ git clone 
2. $ activate venv
3. $ pip install -r requirements.txt

## Extensions
1. Flask-SQLAlchemy
2. Flask-Migrate
3. Flask-Script
4. Flask-OAuthlib
5. mysqlclient
